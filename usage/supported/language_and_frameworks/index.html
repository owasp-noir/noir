Supported Languages and Frameworks

Programming languages and frameworks supported by Noir with feature compatibility details.

Each card below lists a framework's supported features. Route covers endpoint/method/query/path/body/header/cookie/static/websocket plus --include callee callee extraction. AI Context covers the --ai-context buckets: guards, sinks, validators, and signals. Filled chips are supported; struck-through chips are not yet implemented.
