FAQ

OWASP Noir란 무엇인가요?

소스 코드를 정적 분석하여 API 엔드포인트, 웹 페이지 등 공격 표면 진입점을 발견하는 오픈 소스 도구입니다.

Noir를 어떻게 설치하나요?

Homebrew, Snapcraft, Docker 등을 통해 설치할 수 있습니다. 설치 페이지를 참조하세요.

Noir는 어떤 언어와 프레임워크를 지원하나요?

지원되는 언어 및 프레임워크 페이지에서 전체 목록을 확인하세요.

Noir에 어떻게 기여할 수 있나요?

기여 가이드를 읽고 시작하세요.

Noir는 무료인가요?

네, MIT 라이선스의 무료 오픈 소스입니다. 상업적 용도로도 자유롭게 사용할 수 있습니다.

다른 보안 도구와 어떻게 통합하나요?

ZAP, Burp Suite, Caido 등과 통합할 수 있습니다. DAST 파이프라인 가이드를 참조하세요.

결과를 어떻게 내보내나요?

JSON, YAML, OpenAPI 명세, cURL 등 다양한 형식을 지원합니다. 출력 형식 섹션을 참조하세요.

버그 보고나 기능 요청은 어떻게 하나요?

GitHub 이슈를 열어 보고해 주세요.
