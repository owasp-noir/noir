시작하기

이 섹션은 설치부터 첫 번째 스캔까지 OWASP Noir를 시작하는 데 필요한 모든 것을 제공합니다.

세 단계로 Noir를 시작하세요. 이 과정을 마치면 코드베이스를 스캔하고 API 엔드포인트를 발견할 수 있습니다.

단계 페이지 할 것
1 Noir란? Noir가 무엇이고 왜 필요한지 이해하기
2 Noir 설치 시스템에 Noir 설치하기
3 첫 번째 스캔 스캔 실행하고 결과 살펴보기

준비되셨나요? Noir란? 부터 시작하세요.

v0 에서 업그레이드 중이신가요? v0 에서 v1 로 마이그레이션 에 이름 변경 매핑, 동작 변화, 하나뿐인 명시적 비호환 변경이 정리되어 있습니다.
