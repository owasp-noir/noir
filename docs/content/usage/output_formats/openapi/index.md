Generate OpenAPI Specification (OAS) documents from your codebase for API documentation, testing, and tool integration.

Noir supports both OAS 2.0 (Swagger) and OAS 3.0.

## Usage

Generate OAS document:

**OAS 3.0 (recommended)**:

```bash
noir -b . -f oas3
```

**OAS 2.0**:

```bash
noir -b . -f oas2
```
